How to Use:
1. Have 'from_base_to_sql.py' and "IMDB_Dataset_Edited.csv" in the same folder
2. In XAMPP PhpMyAdmin, create a database with name "imdb_moviedb"
3. (highly important) In the config file for MySQL (default path: C:\xampp\mysql\bin\my), change max_allowed_packet to 50M
4. Execute 'from_base_to_sql.py'
